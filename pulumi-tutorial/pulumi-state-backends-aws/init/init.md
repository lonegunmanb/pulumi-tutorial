# State 与 Backend（AWS / MiniStack）

本实验使用 MiniStack 提供一个本地 S3 兼容对象存储，不需要真实 AWS 账号，也不需要 Pulumi Cloud 账号。

你将使用 /root/workspace/state-backends-aws 这个 Pulumi Project，完成这些练习：

- 登录 S3 DIY Backend。
- 创建 dev 与 prod 两个 Stack。
- 观察 State 写入对象存储后的 .pulumi 目录结构。
- 把 Backend URL 写入项目文件。
- 导出并重新导入 Stack State。

点击右侧箭头开始第一步。
